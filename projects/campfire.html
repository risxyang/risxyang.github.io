CAMPFIRE

[Link to Github Repo] / React, Socket.io

Campfire is a collaborative storytelling webapp, built Fall 2020. It is meant to be played as a turn-based game. My co-creators and I developed the idea in quarantine, as we passed time by doing online activities together over group calls. Existing games like Tierlist Maker and Skribble.io served as our inspiration.

Features in the game include: private rooms you can create and join with anybody in the world, and enforced minimum / maximum word limits for both individual turns and the overall story.

Future additions to the game include a UI/UX revamp, export ability for the story, an enforced time limit to each turn (which should make the game aspect of this more challenging and fun), and potentially some NLP features for smart topic suggestions and supplementary graphic generation by the server.
